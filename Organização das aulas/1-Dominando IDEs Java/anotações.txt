sout -para criar o println

control + shift +f10 para roda o programa na primeira vez 

shift +f10 para roda o programa aparti da segunda  vez

ctrl+shift + barra comenta  e descomenta o bloco de codigo selecionado

ctrl + barra comenta  e descomenta o bloco de codigo selecionado linha por linha

Alt+ enter para importar glasse

control + alt +l organiza o codigo

control + D  duplica a linha 

control + Y deleta linha 

control + shift + seta para cima ou para baixo move a linha selecionada
